# BehindYou

基于 YOLO 的实时身后人员检测系统。通过摄像头监控使用者背后区域，当陌生人靠近时自动发送系统通知。

## 功能特性

- **实时人物检测** — 基于 YOLO 目标检测模型，实时识别人物并追踪
- **主人识别** — 通过 InsightFace 人脸特征提取区分主人和陌生人，避免误报
- **智能追踪** — EMA 平滑追踪自身位置，支持快速移动后自动恢复
- **面部验证** — Haar Cascade 前脸检测过滤误检，减少非人物目标干扰
- **系统通知** — 支持 macOS (AppleScript) 和 Linux (notify-send) 原生通知，带冷却时间避免频繁打扰
- **灵活配置** — 支持置信度阈值、检测帧持久性、最小面积、冷却时间等参数自定义

## 快速开始

```bash
pip install -r requirements.txt
python main.py
```

首次运行会自动校准，采集人脸数据。后续启动将快速校准并开始监控。

## 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `--camera` | 0 | 摄像头索引 |
| `--confidence` | 0.6 | 检测置信度阈值 |
| `--cooldown` | 10 | 通知冷却时间（秒） |
| `--persistence` | 3 | 连续检测帧数阈值 |
| `--min-area` | 0.02 | 最小检测框面积占比 |
| `--no-preview` | - | 隐藏预览窗口 |
| `--no-face-check` | - | 禁用人脸验证 |
| `--recalibrate` | - | 强制重新采集人脸数据 |
